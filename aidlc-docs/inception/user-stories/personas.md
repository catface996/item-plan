# 用户画像

## 1. 商家 (Merchant)

| 属性 | 描述 |
|------|------|
| 角色名称 | 商家 |
| 主要职责 | 提交商品信息，响应审核反馈，维护商品数据 |
| 关键目标 | 快速提交商品并通过审核，及时了解审核状态 |

---

## 2. 普通审核员 (Reviewer)

| 属性 | 描述 |
|------|------|
| 角色名称 | 普通审核员 |
| 主要职责 | 执行商品初审，检查商品信息合规性，填写审核意见 |
| 关键目标 | 高效完成初审任务，确保审核质量 |

---

## 3. 高级审核员 (Senior Reviewer)

| 属性 | 描述 |
|------|------|
| 角色名称 | 高级审核员 |
| 主要职责 | 执行商品复审，做出最终审核决定，监督审核质量 |
| 关键目标 | 确保审核决策准确，维护审核标准一致性 |
